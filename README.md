🚀 DiggiTools – Modern Digital Marketplace
DiggiTools is a modern e-commerce web application that provides users with a great experience in shopping for digital products. It is built with the latest technologies like React, Vite, and Tailwind CSS, which makes the app super fast and responsive.

✨ Key Features:
Product Catalog: Browse products from different categories in an attractive grid view.
Smart Shopping Cart: Easily add, increase, or remove products from the cart.
Dynamic Pricing Plans: Different subscription tiers or pricing options for customers.
Fully Responsive: Adapts perfectly to any device—mobile, tablet, or desktop.
Toast Notifications: Instant user feedback for every action (such as adding to cart).
Clean UI: A minimalist and modern user interface built with Tailwind CSS.

🛠️ Tech Stack:
Frontend: React.js (Vite)
Styling: Tailwind CSS
Icons & Assets: Lucide React / FontAwesome
State Management: React Context API / Hooks
